# PillButtonBar

<MediaScroller items={items} />

## Назначение
`PillButtonBar` — контейнерный компонент для **группировки PillButton** и управления выбором между несколькими вариантами. Используется как компактный элемент фильтрации, переключения режимов или выбора контекста внутри экрана.

Компонент визуально представляет собой горизонтальную группу «пилюль», объединённых общей логикой выбора.

---

## Способы использования

### Группа фильтров
Используется для:
- фильтрации контента по категориям;
- быстрого включения/отключения условий;
- локального управления отображением данных.

### Переключение режимов
Используется для:
- смены режимов внутри одного экрана;
- выбора под-разделов без навигационного перехода.

---

## Параметры

### Элементы
- `items` — список элементов группы.
- `selectedItems` — выбранные элементы.
- `selectionMode` — режим выбора (`single`, `multiple`).

### Контент элемента
- `title` — текст кнопки.
- `icon` — иконка (опционально).
- `badge` — числовой или статусный индикатор (опционально).

### Визуальное оформление
- `size` — размер кнопок.
- `style` — визуальный стиль (neutral / brand).
- `spacing` — расстояние между элементами.
- `alignment` — выравнивание группы.
- `wrap` — перенос элементов на следующую строку.

---

## Варианты отображения

### Базовые варианты

1. `Text only`  
   Кнопки содержат только текст.

2. `Icon + Text`  
   Кнопки содержат иконку и текст.

3. `With Badges`  
   Кнопки содержат индикаторы состояния.

---

### Поведенческие варианты

1. `Single Selection`  
   Допускается выбор только одного элемента.

2. `Multiple Selection`  
   Допускается выбор нескольких элементов.

3. `Scrollable Bar`  
   Горизонтальная прокрутка при переполнении.

4. `Wrapped Bar`  
   Перенос элементов на несколько строк.

---

## Состояния (State Machine)

### Базовые состояния
- `Default` — неактивное состояние.
- `Selected` — выбранный элемент.
- `Disabled` — недоступный элемент.

### Состояния взаимодействия
- `Pressed` — состояние нажатия.
- `Focused` — состояние фокуса.

### Модификаторы
- `WithIcons` — используются иконки.
- `WithBadges` — используются бейджи.
- `Scrollable` — включена прокрутка.
- `Wrapped` — включён перенос строк.

---

## Поведение

- Выбор элемента обновляет связанный контент.
- В режиме `single` предыдущий выбор сбрасывается.
- В режиме `multiple` элементы переключаются независимо.
- Размер кнопок адаптируется под контент.
- Компонент корректно реагирует на смену темы.

---

## Ограничения и правила

- Рекомендуемое количество элементов в одной строке — до 5.
- Текст кнопок должен быть коротким и читабельным.
- Не смешивать элементы с разной семантикой.
- Использовать `multiple` только при явной необходимости.
- Компонент не используется для глобальной навигации.

---

## Рекомендации по использованию

Подходит для:
- фильтрации списков и каталогов;
- выбора тегов и категорий;
- управления режимами отображения.

Не рекомендуется для:
- сложной иерархической навигации;
- большого количества вариантов без прокрутки;
- сценариев, где требуется история переходов.
