# SegmentedControl

<MediaScroller items={items} />

## Назначение
`SegmentedControl` — компонент для **переключения между несколькими взаимоисключающими состояниями или представлениями** в рамках одного экрана. Используется, когда требуется быстрое переключение контента без перехода на новый экран.

Компонент располагается внутри контента экрана и управляет отображением связанных данных или режимов.

---

## Способы использования

### Переключение представлений
Используется для:
- смены режима отображения (например, список / карточки);
- фильтрации контента;
- переключения временных периодов или категорий.

### Локальная навигация
Используется как элемент навигации **второго уровня**, не заменяя основные навигационные компоненты.

---

## Параметры

### Сегменты
- `items` — список сегментов.
- `selectedIndex` — активный сегмент.
- `isEnabled` — доступность сегмента.

### Контент сегмента
- `title` — текст сегмента.
- `icon` — иконка сегмента (опционально).
- `badge` — индикатор состояния или количества (опционально).

### Визуальное оформление
- `style` — визуальный стиль (on-brand / on-canvas / on-system).
- `size` — размер сегментов.
- `equalWidth` — равная ширина сегментов.
- `backgroundStyle` — стиль фона контейнера.
- `divider` — разделитель между сегментами.

---

## Варианты отображения

### Базовые варианты

1. `Text only`  
   Сегменты представлены только текстом.

2. `Icon only`  
   Сегменты представлены только иконками.

3. `Icon + Text`  
   Комбинация иконки и текста.

---

### Контекстные варианты

1. `On Canvas`  
   Используется внутри основного контента.

2. `On System`  
   Используется в системных областях.

3. `On Brand`  
   Используется с акцентным брендовым цветом.

---

## Состояния (State Machine)

### Базовые состояния
- `Default` — стандартное состояние.
- `Selected` — активный сегмент.
- `Disabled` — недоступный сегмент.

### Состояния взаимодействия
- `Pressed` — состояние нажатия.
- `Focused` — состояние фокуса.

### Модификаторы
- `WithIcons` — сегменты с иконками.
- `WithBadges` — сегменты с бейджами.
- `Scrollable` — горизонтальная прокрутка сегментов.

---

## Поведение

- При выборе сегмента обновляется связанный контент.
- Активный сегмент визуально выделяется.
- Сегменты имеют равную зону нажатия.
- Переключение происходит без перезагрузки экрана.
- Компонент корректно адаптируется под смену темы.

---

## Ограничения и правила

- Рекомендуемое количество сегментов — от 2 до 5.
- Названия сегментов должны быть короткими.
- Не использовать для сложной иерархической навигации.
- Не смешивать сегменты с разной семантикой.
- Компонент не используется для глобальной навигации.

---

## Рекомендации по использованию

Подходит для:
- фильтрации контента;
- переключения режимов;
- выбора временных периодов;
- локальной навигации внутри экрана.

Не рекомендуется для:
- большого количества вариантов;
- многоуровневой навигации;
- сценариев, где требуется сохранение истории переходов.
